This HTML file contains a pre-order form for "The Steam-iest Summer Sale!" event. 
Customers can use this form to pre-order games, select free games and trial games,
provide delivery details, and choose their preferred payment method.

Files Included:
- index.html: The main HTML file containing the pre-order form.
- Style.css: CSS file for styling the pre-order form.
- Script.js: JavaScript file (currently empty) for any dynamic functionalities.

Coverarts Folder:
- Contains cover images for the games and free/trial games displayed in the form.

Instructions:
1. Open index.html in a web browser to access the pre-order form.
2. Fill out the form with the required information:
   - Customer Information
   - Games Selection
   - Delivery Details
   - Payment Method
3. Click on "Submit" to submit the pre-order or "Reset" to clear the form.
4. Make sure to fill out all required fields before submitting.
